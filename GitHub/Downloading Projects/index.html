Downloading GitHub Projects

This guide will explain to you how to download (clone) GitHub Projects in a editable/collaborative way.

First time GitHub Setup

  1. Download and Install GitHub Desktop
  2. Login to GitHub Desktop

Download

  1. Navigate to the project you want to download on the SuperNURDs GitHub
  2. Click green Code button
  3. Click open with GitHub Desktop
  4. Click clone
  5. Find the download project on your local machine and open files or folders

Committing and Pushing to a Repo

  1. Once you've made edits to your project locally, you can navigate to GitHub Desktop to view your changed files
  2. When you're happy with your set of changes, you can commit them. A commit in Git is essentially creating a snapshot of your changes. To do this, you write a commit message and click the Commit button Note: Committing changes does not upload them. They only live locally on your computer until you push them
  3. Once you've made your commits and are ready to upload your changes, you can click the Push origin button. From there, you and others will be able to see your changes in the GitHub website in browser. Congrats! You've just pushed your changes!
