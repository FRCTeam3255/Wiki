Pit Packing

  1. Set some mallets and fleshcutters asid3
  2. Set aside step ladder
  3. set aside broom and dustpan
  4. Layout all totes in numbered order
  5. Put tubs in totes according to their number
  6. close totes once complete
  7. Set aside boxes without a tote label until later
  8. Get gray dolly and stack all completed totes (2x3 totes)
  9. Put quick grab hardware (i.e. rivets) in tool boxes
  10. Move tool boxes out of pits
  11. Clip shelving zip ties
  12. Make stack of wood shelves
  13. Remove laptop and controllers from driver stations
  14. Pack controllers in special box
  15. Take sign off pit via step ladder (4 person job)
  16. Take down rest of structure
  17. stats box goes in trailer trunk
  18. media stuff stays with media car (tripod goes in trailer trunk)
  19. Sweep up pits
