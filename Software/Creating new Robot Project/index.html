  • Creating new Robot Project
  • Edit on GitHub
  1. Go to New Repository - Standard Swerve Template
  2. Make sure Owner is set to FRCTeam3255
  3. Set Repository name to the format: YYYY_Robot_Code or YYYY_Offbot_Code for example 2024_Robot_Code or 2024_Offbot_Code
  4. Clone the Repo
  5. set merge type restriction
  6. Uncheck all but Allow squash merging

  8. Set the following settings to be checked off

  10. Go to Rules/Rulesets

  11. Select New Rulesets -> Import a ruleset
  13. Download all_branches.json
  14. Download protect_main.json
